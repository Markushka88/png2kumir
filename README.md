# png2kumir
Транслятор картинок формата PNG в формат программы "КуМир" (исполнитель "Черепаха"), сделано от нечего делать в процессе экспериментов по рисованию зеток.

Сделанный файл лучше не загружать в программу, т. к. возникнут проблемы с кодировкой (хотя, может, методом тыка и возможно найти подходящую), так что лучше просто копировать текст получившегося файла, открыв оный, например, в том же **Notepad++**.

Время работы и размер получаемого файла линейно зависит от размера изображения, ну т. е. сложность аппроксимации равна **O(W * H)**.

## Установка

Для начала вам понадобится Python 3.12 (или более новой версии), который вы можете скачать с ![официального сайта](https://www.python.org).

Далее установите библиотеку **Pillow**, если её у вас до сих пор нет:

```
pip install pillow
```

Далее можно запускать сам скрипт.

## Использование

```
python png2kumir.py input.png output.kum
```
